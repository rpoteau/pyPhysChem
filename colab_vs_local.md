## 💻 How to Run This Notebook
There are two ways to use this material, depending on whether you prefer a local setup or a cloud-based environment:

### Option 1: Local Installation

For full control and offline access, you can run this notebook on your own machine. Follow the detailed setup instructions provided in the [README.md](https://github.com/rpoteau/pyPhysChem/blob/main/README.md) file of the main repository to install all necessary dependencies. In short, it consists in:
- Cloning the pyPhysChem repository from GitHub.
- Installing a Python distribution - miniconda is recommended 

### Option 2: Google Colab (Cloud)

If you want to start immediately without installing anything:

- Click the "Open in Colab" button of the notebook.
- Run the first cell (the Setup cell) to automatically configure the environment. This will temporarily download the required files and solutions into the Colab cloud session so that everything functions correctly. It will:
    - Install the required pyphyschemtools library.
    - Clone the necessary repository files (solutions, data, etc.) directly into your Colab session.
    - Set up the working directory so all internal imports work seamlessly.
