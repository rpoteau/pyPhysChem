🍷 Analyse de l'Arbre de Décision
Note technique : Les seuils affichés (ex: -0.89) correspondent à des données centrées-réduites obtenues après application du StandardScaler.
0 représente la moyenne, une valeur négative est inférieure à la moyenne.
  • 1. Le décideur majeur : Le Dioxyde de Soufre (total sulfur dioxide) La racine de l'arbre repose sur le taux de SO₂.
    Observation : Si le taux est très bas (≤ -0.89), le modèle s'oriente vers le vin rouge.
    Interprétation : Les vins blancs nécessitent des doses de sulfites plus élevées pour leur stabilisation et protection contre l'oxydation.
  • 2. Le second critère : Les Chlorures (chlorides) L'algorithme utilise ensuite la concentration en sels.
    Observation : Un taux de soufre bas couplé à des chlorures élevés (> -0.27) mène quasi systématiquement au rouge.
    Interprétation : Les chlorures constituent ici une signature chimique discriminante liée au terroir et aux pratiques de vinification.
  • 3. Zones d'incertitude L'arbre se complexifie quand le soufre et les chlorures sont tous deux très bas.
    Observation : L'algorithme doit alors effectuer un troisième test sur le soufre.
    Interprétation : Cela révèle une zone de recouvrement où les profils des blancs et des rouges "légers" sont très proches.
  • 4. Le "juge de paix" : L'Acidité Volatile (volatile acidity) Ce critère n'intervient qu'en fin de parcours pour les échantillons ambigus.
    Interprétation : L'acidité volatile est typiquement plus élevée dans les vins rouges (processus de fermentation/élevage). L'arbre l'utilise pour affiner la séparation finale.
