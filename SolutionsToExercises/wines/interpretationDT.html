💡 Guide de lecture : Anatomie d'un nœud de l'arbre

Chaque "boîte" (nœud) de l'arbre contient des informations clés pour comprendre la logique de classification :

  • 1. Condition chimique : (ex: chlorides <= -0.272). C'est le critère le plus discriminant à cette étape. Si le vin respecte la condition, on suit la branche de gauche, sinon celle de droite.
  • 2. Indice Gini : Mesure l'impureté du nœud. 0.0 signifie que le groupe est "pur" (une seule couleur). Plus il est proche de 0.5, plus le mélange entre rouge et blanc est équilibré.
  • 3. Samples : Le nombre total de bouteilles de vin analysées et présentes dans ce nœud précis.
  • 4. Value : La répartition exacte des classes. Par exemple, [987, 95] indique la présence de 987 vins rouges et 95 vins blancs.
  • 5. Class : La couleur majoritaire. C'est l'étiquette que le modèle prédirait si l'on s'arrêtait à ce niveau de l'arbre.

💡 Reading guide: Anatomy of a tree node

Each "box" (node) in the tree contains key information for understanding the classification logic:

  • 1. Chemical condition: (e.g. chlorides <= -0.272). This is the most discriminating criterion at this step. If the wine satisfies the condition, we follow the left branch, otherwise the right one.
  • 2. Gini index: Measures the impurity of the node. 0.0 means the group is "pure" (a single color). The closer it is to 0.5, the more balanced the mixture of red and white.
  • 3. Samples: The total number of wine bottles analyzed and present in this specific node.
  • 4. Value: The exact distribution of the classes. For example, [987, 95] indicates the presence of 987 red wines and 95 white wines.
  • 5. Class: The majority color. This is the label the model would predict if we stopped at this level of the tree.
